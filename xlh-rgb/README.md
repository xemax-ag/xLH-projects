# xLH-RGB
Control of the RGB matrix xLH-base. The Python notebooks are executed either via a local Windows 
installation or via a Lite installation on the xLH-base.

## Installation Windows
1. Download GitHub repository
2. Extract the zip file
3. Initialise folder structure: 0_init_folder.bat (once only)
4. Install Python package manager uv: 1_install_uv_package_manager.bat
5. Run the Python notebooks: 2_run_notebooks.bat
