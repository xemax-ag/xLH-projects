# Text
Die Textfunktion ermöglicht es Zeichenketten in eine Leinwand zu integrieren.

putText(img, text, org, fontFace, fontScale, color, thickness=None, lineType=None, bottomLeftOrigin=None)

Powertipp: 
* Mit gedrückter CTRL und einer Bewegung über die Funktion wird der Funktionsheader
  als Tooltip angezeigt. 
* Mit gedrückter CTRL und Klick auf der linken Maustaste über der Funktion wird in die
  Funktionsdatei gesprungen.
